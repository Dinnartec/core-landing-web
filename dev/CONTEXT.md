# Dinnartec Landing — Project Context

## Company

**Dinnartec** is a technology company that builds custom software solutions for businesses, leveraging AI and automation to help clients save time and money.

**Domain:** dinnartec.com
**Repository:** core-landing-web

## Project Overview

This is the main landing page for Dinnartec. The site is bilingual (ES default / EN), responsive, and built with scalability and maintainability in mind.
